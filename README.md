# embarcados-sensor-curta-distancia

Circuito simples que apura uma curta distância (30cm nesse exemplo mas dá pra alterar os valores para 3m mais ou menos) utilizando sensor ultrassônico HC-SR04 com leds indicativos e buzzer utilizando Arduino Uno e outro com Nodemcu.

Circuito simples que detecta palmas utilizando sensor de som KY-038 para ligar leds com uma ou duas palmas utilizando Arduino Uno e outro com Nodemcu.

## Hardware necessário
```
- a) 1 Placa Arduino com cabo USB
- b) 5 Resistores: 300Ω 
- c) 5 LEDs comum de 4mm 
- d) 1 Protoboard
- e) 11-13 Cabos de conexão 
- f) 1 Sensor de som KY-038 
```
